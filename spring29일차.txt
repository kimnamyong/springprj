[이메일 인증받기 절차]

// 환경설정
0) pom.xml에 starter-mail 의존성을 추가한다.
1) 구글에서 SMTP 설정하기 위한 앱 비밀번호를 생성한다.
2) application.properties에 SMTP 비번을 넣는다.
3) SecurityConfig.java 에서 /mail 주소를 permitAll처리한다.
4) csrf를 ignoring 처리한다.

// 서비스 파트
1) memberForm.html에서 인증번호 버튼을 클릭하면 
sendNumber()를 호출하여 ajax에 /mail을 MailController에 
post 매핑요청한다. 
이때 입력한 mail주소가 json 형태로 전송된다.

2) MailController ->MailService로 sendMail() 메소드를 호출한다.

3) MailService에서는 javaMailSender를 사용하여 인증번호를 random()
함수로 생성하고, SMTP로 전송할 이메일,주소,메일본문을 보낸다.

// 프런트단
1) 입력한 이메일에서 받은 인증번호를 확인하여
   memberForm.html 에 입력하고 일치하면 인증완료가 처리된다.
